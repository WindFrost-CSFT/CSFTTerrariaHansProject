## 使用步骤

需要内容：
- 在 Actions 下载最新的 data.unity3d
- 泰拉瑞亚手机版 1.4.5.4.2 版本安装包 apk 文件
- MT 管理器

1. 使用 MT 管理器查看泰拉安装包文件，进入 /assets/bin/Data/ 找到 data.unity3d
2. 把你下载的 data.unity3d 文件添加替换掉原来的那个，压缩级别 ⚠️ 仅储存，选自动签名（注意选错了游戏会出问题）
3. 完成后安装即可，建议备份存档后再安装
